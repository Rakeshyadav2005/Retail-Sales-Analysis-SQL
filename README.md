# Retail Sales Analysis using SQL

## Overview

This project demonstrates the design and implementation of a Retail Sales Database using MySQL. It covers database design, SQL querying, business reporting, and database optimization techniques.

The project simulates a real-world retail environment by analyzing customers, products, and sales transactions.

---

## Project Objectives

* Design a normalized relational database
* Analyze retail sales data using SQL
* Generate business reports
* Implement Views, Stored Procedures, Triggers, and Indexes
* Demonstrate advanced SQL concepts
